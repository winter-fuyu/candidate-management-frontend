### Frontend (React with Tailwind CSS, Ant Design, Tanstack, Axios, and TypeScript)

```markdown
# Candidate Management Frontend

This is the frontend application for the Candidate Management system, built using React, Tailwind CSS, Ant Design, Axios, and TypeScript.

## Features

- View the list of candidates
- Add new candidates
- Update candidate information
- View candidate details

## Technologies

- React
- TypeScript
- Tailwind CSS
- Ant Design
- Axios
- Tanstack Query

## Requirements

- Node.js
- npm or yarn

## Installation

1. Clone the repository:

2. Setup ENV: BACKEND_URL="http://localhost:3001"

3. npm install

## Routes

/ = users list
/new = user create
/update/:id= user update
/:id = user view
```
